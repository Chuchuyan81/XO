# Крестики-Нолики Modern 2.0

Современная веб-реализация классической игры «Крестики-Нолики» с акцентом на дизайн, пользовательский опыт и возможности Progressive Web App (PWA).

## 🎮 Основные возможности

- **Два режима игры**:
  - 👥 **Против друга**: классическое состязание на одном устройстве.
  - 🤖 **Против бота**: игра с алгоритмом, который умеет блокировать ваши ходы и находить победные комбинации.
- **11 уникальных тем оформления**: меняйте стиль интерфейса в один клик. Доступны:
  - *Нейморфизм*, *Глассморфизм*, *Retro (в стиле Windows 95)*, *Dark Mode*, *Material Design*, *Pastel* и другие.
- **PWA (Progressive Web App)**: игру можно установить на компьютер или смартфон как полноценное приложение и запускать без интернета.
- **Система счета**: автоматический учет побед для каждого игрока в рамках текущей сессии.
- **Интерактивный UI**: динамическая отрисовка победной линии, плавные анимации появления символов и модальные окна с результатами.

## 🛠 Технологический стек

Проект написан без использования тяжелых фреймворков, что обеспечивает мгновенную загрузку:
- **HTML5/CSS3**: использование переменных (Custom Properties), Grid и Flexbox для идеальной верстки.
- **Vanilla JavaScript**: вся игровая логика и управление состоянием.
- **Service Workers & Web Manifest**: обеспечивают оффлайн-доступ и возможность установки.
- **LocalStorage**: сохранение выбранной темы оформления при перезагрузке страницы.

## 🚀 Как запустить

Так как проект не требует сборки, запуск максимально прост:
1. Склонируйте репозиторий или скачайте файлы.
2. Откройте файл `index.html` в любом современном браузере.
3. Для полноценной работы PWA рекомендуется использовать локальный веб-сервер (например, расширение Live Server в VS Code).

## 📥 Установка приложения

Для установки игры:
1. Нажмите на иконку меню (три полоски) в верхней части экрана.
2. Выберите пункт **«Скачать приложение»**.
3. После подтверждения иконка игры появится на вашем рабочем столе или в меню приложений.

## 📁 Структура проекта

- `index.html` — основная точка входа, содержащая структуру, стили и логику.
- `manifest.json` — манифест приложения для поддержки PWA.
- `sw.js` — service worker для кеширования и оффлайна.
- `icons/` — набор иконок для различных разрешений экранов.
